# Kanban Board

A responsive Kanban Board application built with HTML, CSS, and JavaScript to help users visually organize and track tasks through different stages of completion.

##  Features

- Create new tasks
- Organize tasks into:
  - To-Do
  - In Progress
  - Done
- Clean and intuitive user interface
- Responsive layout for different screen sizes

##  Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

##  Project Structure

```
kanban-board/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

##  Purpose

This project was built to practice:

- DOM Manipulation

- Frontend Project Structure
